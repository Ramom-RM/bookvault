# 📚 BookVault

O **BookVault** é uma aplicação web desenvolvida para o gerenciamento de livros, permitindo que usuários organizem sua biblioteca pessoal de forma simples e intuitiva.

O sistema conta com autenticação de usuários, pesquisa de livros, listagem de obras e registro de avaliações em estrelas com descrições personalizadas, funcionando como uma biblioteca digital pessoal.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Login de usuários
- Controle de sessão/autenticação
- Listagem de livros cadastrados
- Pesquisa de livros
- Registro de livros no sistema
- Avaliação em estrelas
- Descrição/sinopse personalizada para cada livro
- Armazenamento de dados com SQLite

---

## 🛠️ Tecnologias utilizadas

- **PHP**
- **HTML**
- **CSS**
- **Tailwind CSS**
- **SQLite**

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento web com foco em **backend em PHP**, autenticação de usuários, integração com banco de dados e construção de uma aplicação funcional para portfólio e aprendizado.

---

## ▶️ Como executar o projeto

### Pré-requisitos

- PHP instalado na máquina
- Navegador web
- SQLite habilitado no PHP

### Passos

1. Clone este repositório:

```bash
git clone <URL_DO_REPOSITORIO>
```

2. Acesse a pasta do projeto:

```bash
cd bookvault
```

3. Inicie um servidor local com PHP:

```bash
php -S localhost:8000
```

4. Abra no navegador:

```bash
http://localhost:8000
```
